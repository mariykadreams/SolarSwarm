# User Stories — SolarSwarm Dashboard

## Monitoring

- View live battery level per unit
- See which unit is currently the active/primary light
- See how many hours of coverage remain
- Know whether the system is in grid mode or swarm mode

## Alerts & Events

- Get notified when a unit's battery drops below 30%
- Get notified when a relay (baton pass) happens
- Get notified when all units are critically low (<10%)
- See a log of past events with timestamps

## Control (operators)

- Trigger a grid failure simulation manually
- Reset the demo / restore all units to full charge
- Adjust the demo speed (fast drain for presentations)

## History

- View SOC depletion chart over time
- See relay event history (which unit handed off to which, when)

## Status at a glance

- See unit type (physical vs simulated)
- See last time each unit reported in
- See if a unit is offline/unresponsive
